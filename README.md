# 똑똑
스마트 외출 도우미 똑똑


## 🖥️ 프로젝트 소개
IoT와 안드로이드 앱을 이용한 외출 도우미 시스템

<br>


### 🧑‍🤝‍🧑 맴버구성
 - 팀원 : 김지오 - 로그인, 날씨api, 긴급전화, 일정, DB, 서버, ET-BOARD, ppt 
 - 팀원 : 류시은 - 버스api, 준비물 등록, 발표

### ⚙️ 개발 환경
- **IDE** : Android Studio, 아두이노 IDE(2.3.2)
- **Framework** : Express.js
- **Database** : Mysql(8.3.0)
- **Server** : Node.js (v22.11.0)
